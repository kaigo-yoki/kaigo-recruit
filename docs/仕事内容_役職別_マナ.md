# 仕事内容（役職別・充実版）｜施設訪問介護 マナ

Airワーク採用管理・各求人媒体の「仕事内容」欄にそのまま貼れる原稿です（各1,500〜2,500字程度）。
**1求人＝1役職**で、その役職のブロックだけを貼ってください（給与・雇用形態が役職ごとに違うため）。

前提：マナは**住宅型有料老人ホームに併設した訪問介護（施設内訪問介護）**。入居者様の居室を訪問して介護を提供する形態（移動なし・天候に左右されない・チーム連携しやすい）。

Airワークの職種選択：施設長・サ責→「福祉/介護事業責任者/施設長」／常勤・パート→「介護職／ヘルパー」

## ① 施設長（管理者）

【施設訪問介護「マナ」施設長（管理者）募集｜2027年2月 新規オープン】

有限会社 陽気が2027年2月、沖縄県南城市に新しく開設する住宅型有料老人ホーム併設の訪問介護事業所「マナ」。その立ち上げをリードしていただく施設長（管理者）を募集します。既存のやり方に縛られず、あなたの経験と想いで理想の現場をゼロからつくれるポジションです。

■仕事内容
併設の住宅型有料老人ホームにご入居の方へ、居室を訪問して介護を提供する「施設内訪問介護」事業所の運営全般をお任せします。
・事業所の運営管理（人員配置・稼働管理・収支などの数値管理）
・スタッフの採用・育成・シフト作成・労務管理
・介護保険法等の運営基準の遵守、事故防止・苦情対応・危機管理
・居宅介護支援事業所（ケアマネジャー）、医療機関、ホーム運営側との連携
・サービス品質の管理と、業務ルール・記録体制づくり（オープニングのため一から構築）

■1日の流れ（イメージ）
朝礼・申し送りの確認 → スタッフのシフト・稼働状況チェック → ケアマネや関係機関との連絡調整 → 記録・実績の管理、数値の確認 → スタッフ面談・育成 → 翌日の段取り。現場の状況に応じて、サービス提供のフォローに入ることもあります。

■この仕事の魅力
・移動のない「施設内訪問介護」。天候に左右されず、同じ建物内でチームが連携しやすい環境です
・オープニングだからこそ、運営方針・ルール・チームづくりに最初から関われます
・頑張りが年俸・賞与に反映される業績変動制。数字で成果が見える働き方です

■サポート体制
新規事業所ですが、母体である有限会社 陽気（グループホーム・訪問介護・訪問看護などを運営）のノウハウとバックアップがあります。県外研修（全額会社負担）など学びの機会も豊富です。

■こんな方を歓迎
・数字とロマンのバランスが取れている方／法令理解があり自責で前向きに取り組める方
・教育力（コミュニケーション・マネジメント）のある方／「職員に稼がせたい」という想いのある方

■募集要項
雇用形態：正社員／年俸 500〜600万円（月給350,000〜400,000円・各種手当込み・業績変動制／当社規定のラインをクリアすること）
勤務時間：シフト制（日勤8:00〜17:00／夜勤16:30〜翌8:30 ほか）
休日：週休二日制（基本）／試用期間：1ヶ月（労働条件の変更なし）
待遇：賞与年2回・昇給年1回・企業型確定拠出年金・資格取得補助/県外研修/年2回健診（全額会社負担）・資格手当・通勤手当（上限3,000円）・飲み会/ランチ手当（各月5,000円補助）
応募資格：介護福祉士・実務者研修・初任者研修のいずれか／普通自動車運転免許（AT可）／介護現場でのマネジメント経験者歓迎

まずは見学・お話だけでも歓迎です。

## ② サービス提供責任者（サ責）

【施設訪問介護「マナ」サービス提供責任者募集｜2027年2月 新規オープン】

2027年2月、南城市に新規オープンする住宅型有料老人ホーム併設の訪問介護事業所「マナ」。ご入居の方へのサービスの質を支える、サービス提供責任者を募集します。立ち上げメンバーとして、サービス提供の仕組みづくりから携われます。

■仕事内容
・訪問介護計画書の作成・管理（アセスメント→計画作成→モニタリング→見直し）
・ケアマネジャーとの連携、サービス担当者会議への出席・調整
・訪問介護員（ヘルパー）の指導・育成・技術指導・シフト管理
・利用者様・ご家族との連絡調整、初回訪問・状態把握
・サービス提供体制と品質の管理、記録の整備

■1日の流れ（イメージ）
申し送り確認 → 当日のサービス提供状況・スタッフ配置の確認 → 計画書の作成・見直し、モニタリング → ケアマネ等との連絡調整 → ヘルパーへの指導・相談対応 → 記録整理。必要に応じてご自身も訪問介護を提供します。

■この仕事の魅力
・移動のない施設内訪問介護。同じ建物内でご入居者様の状態を継続的に把握でき、計画的・効率的にサービス提供とマネジメントができます
・オープニングのため、計画書の様式・申し送り・教育の仕組みを一から整えられます
・少人数のフラットなチーム。相談しやすく、意見が反映されやすい環境です

■サポート体制
母体の有限会社 陽気（グループホーム・訪問介護・訪問看護等を運営）のノウハウとバックアップあり。県外研修（全額会社負担）で専門性を高められます。

■こんな方を歓迎
・自責で行動でき、前向きに学び続けられる方
・利用者様・ご家族、スタッフと丁寧にコミュニケーションが取れる方

■募集要項
雇用形態：正社員／年俸 500〜600万円（月給350,000〜400,000円・各種手当込み・業績変動制／当社規定のラインをクリアすること）
勤務時間：シフト制（日勤8:00〜17:00／夜勤16:30〜翌8:30 ほか）
休日：週休二日制（基本）／試用期間：1ヶ月（労働条件の変更なし）
待遇：賞与年2回・昇給年1回・企業型確定拠出年金・資格取得補助/県外研修/年2回健診（全額会社負担）・資格手当・通勤手当（上限3,000円）・飲み会/ランチ手当（各月5,000円補助）
応募資格：介護福祉士、または実務者研修修了者等（サービス提供責任者の要件を満たす方）／普通自動車運転免許（AT可）

## ③ 常勤 介護スタッフ

【施設訪問介護「マナ」常勤 介護スタッフ募集｜2027年2月 新規オープン・夜勤可優遇】

2027年2月オープンの住宅型有料老人ホーム併設 訪問介護「マナ」で、ご入居の方への訪問介護を担う常勤スタッフを募集します。オープニングメンバーとして、一緒に現場をつくっていきましょう。

■仕事内容
併設の住宅型有料老人ホームにお住まいの方の居室を訪問し、介護を提供します。
・身体介護（食事・入浴・排泄・移乗・服薬などの介助）
・生活援助（居室内の環境整備 など）
・入居者様の状態観察、記録、申し送り
・夜勤対応（夜勤可能な方を優遇します）

■1日の流れ（日勤イメージ 8:00〜17:00）
出勤・申し送り → 午前の訪問介護（入浴・排泄介助など）→ 記録 → 休憩 → 午後の訪問介護 → 状態観察・記録 → 申し送り・退勤。夜勤は16:30〜翌8:30のシフトです。

■この仕事の魅力
・移動のない施設内訪問介護。天候に左右されず、同じ建物内で効率よく働けます
・一般の訪問介護のような「1人で抱える不安」が少なく、チームで支え合えます
・オープニングでフラットな人間関係。同期入社が多く、一緒に覚えていけます
・夜勤の回数に応じて年俸が上がり、しっかり稼げます（常勤リーダー手当あり）

■教育・サポート体制
未経験の分野もOJTでサポート。母体の有限会社 陽気のノウハウがあり、資格取得補助金・県外研修はいずれも全額会社負担。働きながらキャリアアップできます。

■こんな方を歓迎
・自責で行動でき、チームワークを大事にできる方
・利用者様に丁寧に向き合える方／体力に自信のある方

■募集要項
雇用形態：正社員／年俸 380〜480万円（月給270,000〜340,000円・夜勤回数で変動・各種手当込み・業績変動制）＋常勤リーダー手当20,000円
勤務時間：シフト制（日勤8:00〜17:00／夜勤16:30〜翌8:30 ほか）
休日：週休二日制（基本）／試用期間：1ヶ月（労働条件の変更なし）
待遇：賞与年2回・昇給年1回・企業型確定拠出年金・資格取得補助/県外研修/年2回健診（全額会社負担）・資格手当（初任者3,000/実務者5,000/介福10,000円）・通勤手当（上限3,000円）・飲み会/ランチ手当（各月5,000円補助）
応募資格：介護福祉士・実務者研修・初任者研修のいずれか／普通自動車運転免許（AT可）

■仕事内容
併設の住宅型有料老人ホームにお住まいの方の居室を訪問し、介護を提供します。
